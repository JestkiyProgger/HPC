# Лабораторная работа 2

Задача: Реализовать алгоритм сложения элементов вектора

Язык: Python

Среда выполнения: Google colab

GPU: NVIDIA Tesla T4

Библиотека CuPy - это библиотека для работы с массивами, совместимая с NumPy/SciPy, предназначенная для ускорения вычислений на графических процессорах с использованием Python. CuPy является полноценной заменой для запуска существующего кода NumPy/SciPy на платформах NVIDIA CUDA или AMD ROCm.

## Проделанная работа

1. Реализация суммирования на CPU:

Для суммирования элементов вектора на CPU использована функция `np.sum()` - суммирует элементы вектора.

2. Реализация суммирования на GPU:

Суммирование на GPU реализовано через библиотеку CuPy, которая предоставляет интерфейс, аналогичный NumPy, но все вычисления производятся на видеокарте с использованием ядер CUDA. Основная операция `cp.sum()` - суммирует элементы вектора. Также используется `cp.asarray()` - преобразует NumPy массив в массив CuPy; `cp.cuda.Stream.null.synchronize()` - синхронизация; `cp.asnumpy()` - преобразует CuPy массив в массив NumPy. 

3. Время вычисления:

Время замеряется с помощью `time.perf_counter()`.

4. Распараллеливание:

Выполняется с помощью ядра CuPy. Каждый блок загружает в память фрагмент вектора, затем внутри блока выполняется параллельная редукция по дереву. Частичные суммы из всех блоков накапливаются либо на CPU, либо с помощью дополнительного ядра.
