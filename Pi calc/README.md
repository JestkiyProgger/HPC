# Лабораторная работа 3

Задача: Реализовать вычисление числа π методом Монте-Карло с использованием CPU и GPU, сравнить производительность.

Язык: Python

Среда выполнения: Google Colab

GPU: NVIDIA Tesla T4

Библиотека PyCUDA - для взаимодействия с CUDA на низком уровне.

## Проделанная работа

1. Реализация на CPU:

Генерация N случайных точек с координатами (x, y) в диапазоне [0, 1] с помощью np.random.rand. Вычисление квадрата расстояния от центра: x² + y². Подсчет точек с x² + y² ≤ 1 с помощью векторных операций NumPy. Вычисление π по формуле: π = 4 * count / N. Измерение времени с помощью time.perf_counter().

2. Реализация на GPU:

Генерация случайных точек на GPU с использованием curandom.XORWOWRandomNumberGenerator(). Запуск CUDA, внутри CUDA ядра каждый поток:
- Загружает свою точку (x, y) из глобальной памяти;
- Проверяет попадание в четверть круга (x² + y² ≤ 1);
- Сохраняет результат (1 или 0) в разделяемую память.

Блочная редукция с использованием разделяемой памяти (последовательное попарное суммирование). Сложение результатов от всех блоков в глобальный счетчик. Синхронизация GPU и копирование результата на CPU.

3. Время вычисления:

Время на CPU измеряется с помощью time.perf_counter(). Время на GPU измеряется с помощью time.time().
