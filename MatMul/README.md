# Лабораторная работа 1

Задача: Реализовать алгоритм перемножения матриц

Язык: Python

Среда выполнения: Google colab

GPU: NVIDIA Tesla T4

Библиотека CuPy - это библиотека для работы с массивами, совместимая с NumPy/SciPy, предназначенная для ускорения вычислений на графических процессорах с использованием Python. CuPy является полноценной заменой для запуска существующего кода NumPy/SciPy на платформах NVIDIA CUDA или AMD ROCm.

## Проделанная работа

1. Реализация умножения на CPU:

Для умножения матриц на CPU использована функция `np.dot()` - cкалярное произведение двух массивов.

2. Реализация умножения на GPU:

Умножение на GPU реализовано через библиотеку CuPy, которая предоставляет интерфейс, аналогичный NumPy, но все вычисления производятся на видеокарте с использованием ядер CUDA. Основная операция `cp.dot()` - Скалярное произведение двух массивов. Также используется `cp.asarray()` - преобразует NumPy массив в массив CuPy; `cp.cuda.Stream.null.synchronize()` - синхронизация; `cp.asnumpy()` - преобразует CuPy массив в массив NumPy. 

3. Проверка корректности перемножения + время вычисления:

Время замеряется с помощью `time.perf_counter()`.

Результаты сравниваются с помощью `np.allclose()` c погрешностью 0,0001.

4. Распараллеливание:

Реализовано через вызов функции cublasSgemm из библиотеки cuBLAS. Эта функция автоматически разбивает результирующую матрицу на блоки, распределяет их между потоковыми блоками CUDA, а внутри каждого блока использует разделяемую память для переиспользования загруженных элементов исходных матриц.
