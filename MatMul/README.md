# Лабораторная работа 1

Задача: Реализовать алгоритм перемножения матриц

Язык: Python

Среда выполнения: Google colab

GPU: NVIDIA Tesla T4

Библиотека CuPy - это библиотека для работы с массивами, совместимая с NumPy/SciPy, предназначенная для ускорения вычислений на графических процессорах с использованием Python. CuPy является полноценной заменой для запуска существующего кода NumPy/SciPy на платформах NVIDIA CUDA или AMD ROCm.

## Проделанная работа

1. Реализация умножения на CPU:

Для умножения матриц на CPU использована функция `np.dot()`.

2. Реализация умножения на GPU:

Умножение на GPU реализовано через библиотеку CuPy, которая предоставляет интерфейс, аналогичный NumPy, но все вычисления производятся на видеокарте с использованием ядер CUDA. Основная операция `cp.dot()`. Также используется `cp.asarray` - преобразует NumPy массив в массив CuPy; `cp.cuda.Stream.null.synchronize()` - синхронизация; `cp.asnumpy` - преобразует CuPy массив в массив NumPy. 

3. Проверка корректности перемножения + время вычисления:

Время замеряется с помощью `time.perf_counter()`.

Результаты сравниваются с помощью `np.allclose(C_cpu, C_gpu_cpu, rtol=1e-4, atol=1e-4)`.

4. Распараллеливание:

Распараллеливание осуществляется автоматически библиотекой CuPy, которая использует CUDA для запуска большого числа потоков, обрабатывающих элементы матрицы параллельно.
